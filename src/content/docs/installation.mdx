# Installation

Get started with Atomic UI in your Next.js project.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js 18+**
- **npm**, **yarn**, or **pnpm**
- A Next.js 13+ project with **app directory**

## Manual Installation

### 1. Install Dependencies

First, install the required dependencies:

```bash
npm install clsx tailwind-merge class-variance-authority
npm install -D @types/react @types/react-dom
```

### 2. Configure Tailwind CSS

Make sure your `tailwind.config.ts` includes the following:

```ts
import type { Config } from "tailwindcss";

const config: Config = {
  darkMode: ["class"],
  content: [
    "./src/pages/**/*.{js,ts,jsx,tsx,mdx}",
    "./src/components/**/*.{js,ts,jsx,tsx,mdx}",
    "./src/app/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    container: {
      center: true,
      padding: "2rem",
      screens: {
        "2xl": "1400px",
      },
    },
    extend: {
      colors: {
        border: "hsl(var(--border))",
        input: "hsl(var(--input))",
        ring: "hsl(var(--ring))",
        background: "hsl(var(--background))",
        foreground: "hsl(var(--foreground))",
        primary: {
          DEFAULT: "hsl(var(--primary))",
          foreground: "hsl(var(--primary-foreground))",
        },
        secondary: {
          DEFAULT: "hsl(var(--secondary))",
          foreground: "hsl(var(--secondary-foreground))",
        },
        destructive: {
          DEFAULT: "hsl(var(--destructive))",
          foreground: "hsl(var(--destructive-foreground))",
        },
        muted: {
          DEFAULT: "hsl(var(--muted))",
          foreground: "hsl(var(--muted-foreground))",
        },
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
        popover: {
          DEFAULT: "hsl(var(--popover))",
          foreground: "hsl(var(--popover-foreground))",
        },
        card: {
          DEFAULT: "hsl(var(--card))",
          foreground: "hsl(var(--card-foreground))",
        },
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
        sm: "calc(var(--radius) - 4px)",
      },
    },
  },
  plugins: [],
};

export default config;
```

### 4. Create Utils File

Create `src/lib/utils.ts`:

```ts
import { type ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

## Using the CLI (Recommended)

The easiest way to add components is using our CLI tool:

### Install a Component

```bash
npx my-ui add gradient-button
```

This will:
- Download the component files
- Install required dependencies
- Add the component to your project

### List Available Components

```bash
npx my-ui list
```

### Add Multiple Components

```bash
npx my-ui add gradient-button shiny-card button
```

## Troubleshooting

### TypeScript Errors

If you're getting TypeScript errors, make sure:
- You have the latest version of TypeScript
- Your `tsconfig.json` includes the correct paths
- All required dependencies are installed

### Styling Issues

If components don't look right:
- Verify Tailwind CSS is properly configured
- Check that CSS variables are defined
- Ensure dark mode is set up if needed
